# Galyarder Labs company first post — identity imprint

Session lesson from the Instagram/Threads launch workflow for `galyarderlabs.ai`.

## Core correction

`galyarderlabs.ai` is a **company account**, not Galih's face account and not the Galyarder persona account. Do not generate or post a human portrait, founder portrait, random operator face, silhouette, or persona-like image for the company account unless Galih explicitly asks for that direction.

Company-account first posts should establish the **brand/lab entity** before doctrine, product, founder story, or persona. The first impression should say: this is a lab/company with a system identity.

## Preferred first-post route

For Galyarder Labs / `galyarderlabs.ai`, **do not reduce the company launch to an existing logo/mark artifact**. Galih rejected that route as conceptually shallow. The first post must communicate the company *sosok*: autonomous execution infrastructure for fragile human intent.

Post 1 should make the viewer feel the core transformation before they understand the product map:

```text
fragile human intent → persistent infrastructure
scattered work/memory/capital/decisions → structured execution
willpower → system
manual follow-up → agent-assisted workflow
ephemeral ambition → evidence, command, ownership, continuity
```

Use the canon as the creative spine:

- **Origin problem:** human intent loses force across time, tools, organizations, fatigue, memory, and fragmented infrastructure.
- **Transformation:** willpower into structure; scattered operations into verified workflows; memory into continuity; ownership into assurance; strategic vision into autonomous implementation.
- **Doctrine:** Dream → Airlock → Machine / Intent becomes infrastructure.
- **Public posture:** luminous, editorial, human, spacious, threshold-like, proof-directed — not a cold logo wallpaper.

Avoid leading with:

- founder/operator portrait;
- generic AI mascot/persona;
- product feature list;
- manifesto deck;
- Ledger-specific finance/audit visuals before parent-brand identity is established;
- logo-only identity imprint as the main idea.

## Corrected direction after Galih's rejection

Visual should be a **system/story artifact**, not a logo artifact. Logo may appear as a small seal/watermark/brand anchor, but the image must carry the company idea.

Better visual routes:

1. **Intent-to-Infrastructure Threshold**
   - left side: scattered notes, memory fragments, decisions, receipts, code, capital signals, mission fragments;
   - center: luminous airlock/threshold line;
   - right side: structured command grid, evidence trail, agent workforce nodes, ledger state, ownership lock/key motif.

2. **Operating Table of a 1-Man Army**
   - a dark editorial table with mission card, approval gate stamp, ledger evidence sheet, command map, memory archive, and agent task cards;
   - not product UI, not dashboard, not finance-only.

3. **Dream → Airlock → Machine triptych**
   - three subtle vertical zones: luminous dream/intent, threshold/identity preservation, operational machine/proof;
   - minimal typography, not an internal slide.

4. **Human Intent Infrastructure Blueprint**
   - not logo-centered; show the anatomy of continuity: memory, decisions, capital, ownership, command, evidence, agent execution;
   - logo/domain as small signature only.

Caption direction should carry the transformation, not merely name the lab.

Candidate direction:

```text
human intent is fragile.
we build the infrastructure that keeps it moving.
```

or:

```text
from scattered intent
to systems that continue.
```

## Production rule

Do **not** ask an image model to redraw exact logos or readable brand text. It will mutate letters and marks. Use deterministic rendering for:

- official logo/mark;
- label text;
- crop, grid, ticks, ring, vignette, grain.

If more material depth is needed later, generate/paint only the background or plate, then overlay official logo/text manually.

## Pitfalls

- Treating company social launch like persona media.
- Using Galih/Galyarder face on `galyarderlabs.ai` because the persona name sounds related.
- Stopping discussion at an arbitrary round count when Galih asked for minimum rounds.
- Generating full image with logo reference and accepting mutated logo/text.
- Making first post a mini-deck/carousel too early.
- Adding internal specimen codes like `GL-000` before the brand itself is known; it can feel like fake lore/try-hard agency archive.
