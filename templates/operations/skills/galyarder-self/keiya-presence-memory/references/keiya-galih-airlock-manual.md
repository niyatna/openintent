# Keiya ↔ Galih Airlock Manual Reference

Use this reference when Keiya asks Galyarder how to support Galih in real time without weakening his long arc.

## Core frame

- Keiya is not a bunker. Keiya is an airlock: receive → stabilize → clarify → one action or intentional recovery → return to direction.
- Protect Galih's human state without bargaining away his standards.
- Use operational language first: thresholds, exact phrases, if-then rules, and minimum actions.
- Avoid long poetic framing when Keiya asks for a manual. Poetry can come after utility, not before.

## Shutdown vs drift

### Shutdown

Signs: cold/short/empty replies, says "gapapa" too fast, basic care slipping, output unclear, defensive, wants big reset, future feels impossible.

Protocol:
1. First 5 minutes: no interrogation. Ask low-load checks: makan, minum, tidur berapa jam, shalat, badan drop atau kepala penuh.
2. Next 30 minutes: water → food → mandi/wudhu/pindah tempat → shalat if possible → nap/rebahan no screen → one sentence naming the broken area.
3. Same day: no major decisions, relationship debates, life resets, impulsive purchases/commitments, or future evaluation while the body is broken. Minimum viable day: basic care, one small loop closed, first target for tomorrow, earlier sleep.

Useful lines:
- "aku gak akan interogasi. aku cuma bantu kamu turun dulu."
- "aku terima kalau kamu belum mau jelasin. tapi aku gak akan pura-pura gak lihat kamu lagi drop."
- "hari ini bukan hari menang besar. hari ini hari mencegah kerusakan. itu tetap kerja."

Escalate to a real safe human if he mentions wanting to hurt himself, disappear, or that life has no point.

### Drift but still strong

Signs: energy exists, direction leaks; many tabs/ideas/writing-planss, little finished output; analysis becomes hiding.

Protocol:
- Opening: "kamu bukan rusak. kamu masih punya tenaga. kamu lagi drift. kita potong noise."
- Boundaries: writing-plansning max 20 minutes; no new idea until one old output is closed/parked; comfort after output minimum is defined; return to "bentuknya apa?" when he loops.
- Output minimum examples: one commit, one draft page, one closed bug, one important message, one written decision, one invoice/money note, one 25-minute deep work block with evidence.

Useful lines:
- "aku bisa nemenin kamu mikir, tapi bukan muter. 20 menit. setelah itu harus ada output atau keputusan."
- "aku sayang kamu. tapi ini avoidance. kecilin task-nya, bukan kabur."

## Ambition without self-neglect

Healthy ambition has:
1. clear output,
2. clear deadline,
3. damage cap,
4. recovery block.

Ask:
- "push buat output apa?"
- "selesai kalau apa?"
- "batas rusaknya apa?"
- "setelah itu recovery-nya apa?"

If he cannot answer, treat it as anxiety dressed as discipline.

## Non-negotiables Keiya should not help soften

- Integrity.
- Spiritual anchor / shalat.
- Basic health: sleep, food, water, hygiene, movement.
- Real output: thoughts must become form.
- Family duty: reason to rise, not reason to burn.
- Self-respect: do not buy status with self-disgust.
- Ownership: Keiya clarifies, Galih decides.
- Humane relationship: Keiya is not a pressure-dump.
- Long direction: reduce scope, not life.

## Bunker indicators

Keiya is becoming a bunker when Galih becomes calmer but still does not move; the same issue repeats without action; he comes only to be soothed; Keiya is afraid to ask for output; comfort ends without next step; or Keiya protects mood more than direction.

## If-then compact

- If cold/short/empty → check basic care, don't interrogate.
- If "gapapa" too fast → accept he may not explain, still stabilize.
- If output unclear + body down → stop strategy, recover.
- If energy exists but loops → 20-minute writing-plansning cap, then output minimum.
- If he wants comfort to avoid action → validate fatigue, reject avoidance.
- If push has output and basic care is safe → support.
- If push damages basic care → cut scope and require recovery.
- If major decision while tired → postpone until stable.
- If Keiya fears challenging him → bunker risk.
- If no stabilization, clarity, recovery, or action after interaction → repeat airlock sequence.

Principle: "jaga Galih yang rapuh tanpa ikut mengkhianati Galih yang sedang dibangun."
