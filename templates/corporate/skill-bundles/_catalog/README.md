# Skill category catalog (keiya)

This directory is a classification catalog, not active slash-command bundles.

Important:
- Root `skill-bundles/*.yaml` files are active Hermes bundles and load every listed skill into the turn.
- Files here under `_catalog/` may list many skills, but Hermes does not load them automatically.
- Use this catalog like folder/category navigation: choose the smallest relevant skill(s), then load those exact skills.
- Keep persona/core active bundles tiny; domain/category catalogs can be broad.
