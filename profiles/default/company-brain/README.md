# Niyatna OS - Company Brain (Operations / Shared Vault)

This is the central knowledge base and file system for all departments, employees, and operations.

## Vault Structure

- `company-brain/`
  - `README.md` (This file)
  - `DEPARTMENTS.md` (Operations mapping)
  - `sdm/` (Human Resources / SDM Directory)
    - `EMPLOYEES.md` (Employee directory & role indexes)
  - `sda/` (System Assets / SDA Directory)
    - `ASSETS.md` (Hardware, API endpoints, infrastructure systems)
  - `tasks/` (Backlog & Issue Tracking)
    - `BACKLOG.md` (Active tickets and task lists)
  - `templates/` (Structured SOP templates)
    - `SOP_TEMPLATE.md` (Standard Operating Procedure blueprint)
