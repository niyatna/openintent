# Niyatna OS - Department Directory (Company OS)

This document maps all active company departments and their operational lines.

## 1. Operations (OPS)
- **Role**: Core systems management, codebase operations, container verification, and server audit checks.
- **Data Owner**: Operations Agent / Default Profile.
- **Workflow**: Mapping task backlogs directly to Paperclip visual boards.

## 2. Human Resources (SDM)
- **Role**: Hiring pipelines, team onboarding, habits tracking, and employee role allocations.
- **Workflow**: Managed via `/company-brain/sdm/` folder directory.

## 3. Product & Technology (IT/SDA)
- **Role**: System assets monitoring, API health integration checks, and tech stack orchestration.
- **Workflow**: Managed via `/company-brain/sda/` folder directory.

## 4. Sales & Growth (CMO/Marketing)
- **Role**: Competitor stock intelligence logs, client-facing materials, and public branding packages.
- **Workflow**: Connects to the Public Agent's public FAQ matrices.
